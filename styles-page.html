Customizing a Bootstrap website with Sass

Variables

Bootstrap offers ways to override the built-in variables. Keep in mind that the override must occur before importing Bootstrap.

          
  $body-bg: #1a1a1a;
  $body-color: #eee;

  // Bootstrap and its default variables
  @import "../node_modules/bootstrap/scss/bootstrap";

Variables

All the variables can be found in /node_modules/bootstrap/scss/_variables.scss

Custom Styles

I've added more styles after the import to override them. I used this structure to set the dark color scheme.

          
  // Override values
  $body-bg: #1a1a1a;
  $body-color: #eee;

  // Global Styles
  .navbar {
    background-color: black;
  }

  // Bootstrap and its default variables
  @import "../node_modules/bootstrap/scss/bootstrap";

  // More styles here
  .alert-info {
    background-color: #010c21;
    color: #8cc4e0;
  
    .alert-link {
      color: #b6dce3;
    }
  } // etc...

Mixins

Here's an example of a specific styling rules for the .sidenav for widths up to 767px.

          
  @media (max-width: 767px) {
    .sidenav {
      border: 1px solid rgba(255, 255, 255, 0.2);
      margin: 20px 0;
      border-radius: 5px;
      background-color: rgba(255, 255, 255, 0.03);
    } 
  }

Alternatively, when using Bootstrap's media queries using mixins, it looks like this:

          
  @include media-breakpoint-down(sm) {
    .sidenav {
      border: 1px solid rgba(255, 255, 255, 0.2);
      margin: 20px 0;
      border-radius: 5px;
      background-color: rgba(255, 255, 255, 0.03);
    } 
  }

Caution

These type of rules must be written after the import statement. Otherwise, you will get an error.

For example:

            
  Error: Undefined mixin.
