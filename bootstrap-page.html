Bootstrap

Bootstrap is a CSS framework for building responsive mobile first websites.

Layout

This website consists of the top navigation bar and the content div with a .container class.

The .container class has multiple max-width at each breakpoints.


  <body>
    <nav class="navbar fixed-top navbar-expand-sm navbar-dark"></nav>
    <div class="container"></div>
  </body>

The .container element is a parent to these two child elements, left sidenav and the main container to the right.


  <div class="container">
    <div class="row">
      <div class="col-md-3">
        <nav class="sidenav" aria-label="Side Nav"></nav>
      </div>
      <main class="col-md-9 py-md-3" role="main">
      </main>
    </div>
  </div>

The column widths are col-md-3 and col-md-9 for the sidenav and the main container, respectively.

For clarification, this table from the documentation shows the class prefixes and their widths fairly well:

.py-md-3 and .pl-md-5 can be broken down like this {property}{sides}-{breakpoint}-{size}

  • p - for classes that set padding
  • y - for classes that set both *-top and *-bottom
  • l - for classes that set margin-left or padding-left
  • 3 - (by default) for classes that set the margin or padding to $spacer
  • 5 - (by default) for classes that set the margin or padding to $spacer * 3
Refer to the Bootstrap documentation for more information

Navigation Bar

The navigation bar is responsive and collapses at 576px of viewport width, defined by .navbar-expand-sm

            
  <nav class="class="navbar fixed-top navbar-expand-sm navbar-dark bg-dark"">
    <a class="navbar-brand" href="./index.html">
      <img src="./images/home-white.svg" alt="home icon">
    </a>
    <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarSupportedContent" aria-controls="navbarSupportedContent" aria-expanded="false" aria-label="Toggle navigation">
        <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarSupportedContent">
        <ul class="navbar-nav mr-auto">
            <li class="nav-item"><a class="nav-link" href="./index.html">Home</a></li>
            <li class="nav-item"><a class="nav-link" href="./gettingstarted-page.html">Getting Started</a></li>
            <li class="nav-item active"><a class="nav-link" href="./bootstrap-page.html">Bootstrap</a></li>
            <li class="nav-item"><a class="nav-link" href="./styles-page.html">Styles</a></li>
        </ul>
        <hr />
        <ul class="navbar-nav">
            // list all social links here
        </ul>
    </div>
  </nav>

Since the nav bar is fixed to the top using .fixed-top, I needed to add .padding-top to the body to prevent overlap of content.

            
  body {
    padding-top: 56px;
  }

Warning

Due to this padding on the body, jumping with anchor link didn't position the webpage correctly. Applying this fix to all anchor headers solved the issue for me:

              
    <h1 style="padding-top: 70px; margin-top: -70px;">Bootstrap</h1>
